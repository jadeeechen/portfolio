# jadeeechen.com

Personal portfolio website for Jade Chen — data scientist and MDS student at UBC.

Built with [Next.js](https://nextjs.org/) and [TailwindCSS](https://tailwindcss.com/), deployed on [Netlify](https://netlify.com/).

## Running locally

```bash
yarn
yarn dev
```

## Based on

[react-portfolio-template](https://github.com/chetanverma16/react-portfolio-template) by Chetan Verma
